# CreativeMode
Bephinex plugin that uses harmony patching and a custom UI to add "creative mode" (like from minecraft) fetaures to 20 mins till dawn.

To download the mod only, take the "CreativeMode" folder and put it into the plugins folder from bephinex. Or download it from thunderstore (recommended).

# Controls
press tab to toggle the UI and pause/unpuase the game.
